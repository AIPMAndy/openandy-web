## 1. 产品定位

### 1.1 核心定位
**产品名称：** AI 酋长 - 东方智慧决策系统  
**Slogan：** 千年智慧，AI 赋能，为决策者而生  
**核心价值：** 为 CEO/企业家提供基于东方智慧的人生重大决策洞察

### 1.2 目标用户
- **主要用户：** CEO、企业家、高净值人群
- **年龄段：** 35-55 岁
- **痛点：** 面临重大决策时缺乏深度洞察和智慧指引
- **需求：** 个人决策辅助（职业发展、时间管理、人生规划）

### 1.3 产品战略
- **短期目标：** 通过"天机测算"功能吸引目标用户，建立品牌认知
- **中期目标：** 引流到 AI 酋长 Andy 1对1 深度咨询
- **长期目标：** 打造高端决策辅助服务品牌，建立企业家社群

### 1.4 商业模式
- **收费方式：** 按次付费
- **定价策略：** 三档定价
  - 基础测算：¥299（AI 自动生成）
  - 深度解读：¥999（AI + 专家点评）
  - 至尊咨询：¥2999（Andy 亲自解读 + 1对1 咨询）

│  ├─ 三大核心价值

│  │  └─ 决策者专属（CEO、企业家定制）

## 3. 视觉设计系统

### 3.1 配色方案（东方美学）

**主色调：**
- **墨色（深灰黑）：** `#1A1A1A` - 主文字、标题
- **宣纸白（暖白）：** `#FAF9F6` - 背景
- **朱砂红（点缀）：** `#C8102E` - CTA 按钮、重要信息

**辅助色：**
- **水墨灰：** `#6B6B6B` - 次要文字
- **淡墨色：** `#D4D4D4` - 边框、分割线
- **金色（高端感）：** `#D4AF37` - 价格、VIP 标识

### 3.2 字体系统

**中文字体：**
- **标题：** 思源宋体 / 方正清刻本悦宋（优雅、传统）
- **正文：** 苹方 / PingFang SC（现代、易读）

**英文/数字：**
- **标题：** Playfair Display（优雅衬线）
- **正文：** Inter（现代无衬线）

### 3.3 视觉元素

**背景纹理：**
- **首页：** 水墨渐变效果（从深到浅）
- **测算页：** 淡雅的宣纸纹理
- **结果页：** 水墨晕染边框

**图标风格：**
- 中国风线性图标
- 简约、禅意

**动效：**
- **页面切换：** 水墨晕染过渡
- **按钮悬停：** 淡入淡出
- **加载动画：** 太极旋转或水墨扩散

### 3.4 组件设计

**按钮：**
- **主按钮：** 朱砂红背景 + 白色文字 + 圆角 8px
- **次按钮：** 墨色边框 + 墨色文字 + 圆角 8px

**卡片：**
- 白色背景 + 淡墨色边框
- 圆角 12px
- 轻微阴影（禅意感）

**表单：**
- 简洁输入框
- 墨色边框
- 圆角 8px
- 聚焦时朱砂红边框

**测算记录表（fortune_readings）：**

CREATE TABLE fortune_readings (
  id VARCHAR(36) PRIMARY KEY,

  user_info JSON,

  created_at TIMESTAMP,
  FOREIGN KEY (order_id) REFERENCES orders(id)
);

**里程碑 2：** 核心功能完成（7 小时后）
- 套餐选择功能

### 6.3 风险与应对

**风险 1：支付集成复杂度超预期**
- **应对：** 先实现模拟支付，后续补充真实支付

**风险 2：视觉效果实现困难**
- **应对：** 简化动效，保证核心视觉风格

**风险 3：时间不足**
- **应对：** 优先完成 P0 任务，P1 任务可后续迭代
